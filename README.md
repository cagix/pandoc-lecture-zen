This project defines a GitHub action to create slides and handouts (PDF) and GitHub Markdown
(GitHub preview) from Markdown files using [Pandoc].

It's possible that this will succeed the [Pandoc-Lecture] Project, or that this will be
integrated into the [Pandoc-Lecture] Project at some point.
